## 기획의도

사용자에게 다량의 정보를 받을때 '어떻게 해야 효과적일까' 라는 고민에서 시작

```
 - 첫째. 지루해 보이지 않아야 한다
 - 둘째. Simple is the best
 - 셋째. 한눈에 보여야 한다 ( 다른 뷰로 넘어가지 않는 방법)
```

위 세가지를 중점으로 빌드업
